# RFM-E-Commerce_Analysis
## Setup environment
```bash
conda create --name main-ds python=3.9
conda activate main-ds
pip install numpy pandas scipy matplotlib seaborn jupyter streamlit babel plotly
```

## Run steamlit app
```bash
cd Dashboard
streamlit run dashboard.py
```
